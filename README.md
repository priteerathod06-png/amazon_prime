# amazon_prime
# 📺 Amazon Prime Video Dashboard | Power BI

## 📌 Project Overview

This Power BI dashboard provides an interactive analysis of the Amazon Prime Video dataset. It helps users explore content based on movies, TV shows, genres, ratings, release years, countries, and directors. The dashboard enables quick insights into Amazon Prime's content library using interactive visualizations and KPIs.

---

## 🎯 Objectives

- Analyze Amazon Prime Video content.
- Compare Movies and TV Shows.
- Identify the most popular genres.
- Analyze content ratings.
- Visualize release trends over the years.
- Explore content distribution by country.
- Build an interactive dashboard for better business insights.

---

## 📊 Dashboard Features

- ✅ Total Titles KPI
- ✅ Total Directors KPI
- ✅ Total Genres KPI
- ✅ Total Ratings KPI
- ✅ Movies vs TV Shows Distribution
- ✅ Content by Country (Map)
